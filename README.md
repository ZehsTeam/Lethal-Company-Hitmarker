# Hitmaker
Shows a hitmarker when you successfully hit an enemy. With additional features. (Client-side)

<ins><b>All features are configurable in the config settings.</b></ins>

* Shows a hitmarker when you successfully hit an enemy.
* Shows a message of how much damage you did to an enemy.
* Shows a message when you or another player kills an enemy.
* Fully client side. (Works in vanilla lobbies)

Download [Hitmarker](https://thunderstore.io/c/lethal-company/p/Zehs/Hitmarker/) on Thunderstore.

## Config Settings
<details>
  <summary>Expand</summary>
<br>

| General Settings | Setting type | Default value | Description |
| ----------- | ----------- | ----------- | ----------- |
| `ExtendedLogging` | `Boolean` | `false` | Enable extended logging. |

| Hitmarker Settings | Setting type | Default value | Description |
| ----------- | ----------- | ----------- | ----------- |
| `ShowHitmarkerImage` | `Boolean` | `true` | Do you want to show the hitmarker image? |
| `HitmarkerImageSize` | `Int32` | `40` | The size of the hitmarker image in pixels. |
| `PlayHitmarkerSound` | `Boolean` | `true` | Do you want to play the hitmarker sound? |

| Message Settings | Setting type | Default value | Description |
| ----------- | ----------- | ----------- | ----------- |
| `MessageDuration` | `Single` | `4` | The message duration in seconds. |
| `MessageFontSize` | `Single` | `35` | The message font size in pixels. |
| `ShowDamageMessage` | `Boolean` | `true` | Shows a message of how much damage you did to an enemy. |
| `ShowKillMessage` | `Boolean` | `true` | Shows a message when an enemy is killed. |
| `OnlyShowLocalKillMessage` | `Boolean` | `true` | Will only show your kill messages. |

</details>
